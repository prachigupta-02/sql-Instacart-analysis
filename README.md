# sql-Instacart-analysis
This project analyzes the Instacart dataset to reveal product performance and sales trend using SQL.

## dataset
The dataset used is the Instacart Online Grocery Dataset (2017), which contains anonymized data on customer orders, products, aisles, and departments.

## 📊 Analysis Performed
- Product Analysis
- Customer Behavior Analysis
- Sales Trend Analysis
- Category Analysis

## Key Insights
- Customers tend to reorder grocery items, indicating strong product loyalty towards it.
- Some departments and aisles contribute significantly high to total purchases made by consumers such as beverages-soft drinks.
- The average number of products per order suggests that customers often purchase multiple items in a single transaction.
- Customer segmentation reveals mostly customers have ordering frequency in the interval of 8-20 days.
- Order activity is not evenly distributed and shows a clear variation across days of the week, Monday being at the top.
